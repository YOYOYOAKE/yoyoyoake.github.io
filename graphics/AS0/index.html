作业描述：

给定一个点P=(2,1)P=(2,1)P=(2,1)，将该点绕原点先逆时针旋转45°45°45°，再平移(1,2)(1,2)(1,2)，计算出变换后点的坐标（要求使用齐次坐标进行计算）.

1 数学思路

首先写出点PPP的齐次坐标(2,1,1)(2,1,1)(2,1,1).

然后写出旋转矩阵

R=(cos⁡45°−sin⁡45°0sin⁡45°cos⁡45°0001)R = \begin{pmatrix} \cos 45° & -\sin 45° & 0 \\ \sin 45° & \cos 45° & 0 \\ 0 & 0 & 1 \end{pmatrix} R=​cos45°sin45°0​−sin45°cos45°0​001​​

然后写出平移矩阵

T=(101012001)T = \begin{pmatrix} 1 & 0 & 1 \\ 0 & 1 & 2 \\ 0 & 0 & 1 \end{pmatrix} T=​100​010​121​​

于是变换后的坐标为

P′=T⋅R⋅PP' = T \cdot R \cdot P P′=T⋅R⋅P

2 程序设计

int main()
{
    Eigen::Vector3f p(2, 1, 1), pNew;

    Eigen::Matrix3f r, t;

    // 定义旋转角
    float rad = 45.0 / 180.0 * M_PI;

    r << cos(rad), -sin(rad), 0,
        sin(rad), cos(rad), 0,
        0, 0, 1;

    t << 1, 0, 1,
        0, 1, 2,
        0, 0, 1;

    pNew = t * r * p;

    std::cout << "The new homogenous coordinate is " << std::endl;
    std::cout << pNew << std::endl;
  
    return 0;
}
